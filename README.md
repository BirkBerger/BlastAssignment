## Getting Started

# CS:GO Game Stats

A Next.js app built as part of a coding challenge for [Blast](https://blast.tv/).
It shows scoreboard, player cards and game chart.

Built by **Thea Birk Berger**.

## Tech Stack
- Next.js 16, React, TypeScript

Requires Node 20+

To get project

```bash
git clone https://github.com/BirkBerger/BlastAssignment.git
cd BlastAssignment
```

Create file .env.local in root
```bash
NEXT_PUBLIC_API_URL=http://localhost:3000
STEAM_API_KEY=<your-steam_api-key>
```

You can find your Steam Api-key [here](https://steamcommunity.com/dev/apikey).
Filling it out is optional but recommended.

To run project

```bash
npm install
npm run dev
```

Open http://localhost:3000 with your browser to see the result.
